How to reach the full interface

Build the React frontend

From the project directory, run:

cd frontend

npm install

npm run build

Then reopen the address printed by vulnclaw web.

Safety boundary

Do not cross the authorized scope

The full web interface renders only_*, blocked_*, and action constraints as a readable boundary, and shows the reason on the safety-boundary page whenever an out-of-scope attempt is blocked.

For routine self-testing, specify at least one of host, port, or path.

If you have already built the frontend but still see this page, confirm that frontend/dist/index.html exists and restart the web service.
